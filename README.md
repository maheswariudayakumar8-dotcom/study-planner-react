# Study Planner React

A React + Vite project built for Mission 3 of the VertexMind Internship.

## Features

* React Components (Navbar, Card, Footer)
* React Router Navigation
* Home Page
* About Page
* Project Page
* useState Counter Feature
* Responsive Design

## Tech Stack

* React
* Vite
* React Router DOM
* JavaScript
* CSS

## Run Locally

```bash
npm install
npm run dev
```
